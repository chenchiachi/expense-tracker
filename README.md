# 家庭記帳本

## 基本功能
- 在首頁一次瀏覽所有支出的清單

- 在首頁看到所有支出清單的總金額
  
- 新增一筆支出

- 編輯任何一筆支出的所有屬性

- 刪除任何一筆支出

- 在首頁可以根據支出「類別」篩選支出廳


## 安裝
- git clone  此專案
```
https://github.com/chenchiachi/expense-tracker.git
```
- cd 至專案資料夾
- 安裝 npm 套件
- 使用 npm 開啟程式
